# Casoola Casino Landing Page

A landing page for Casoola Casino, featuring an Art Deco Grand Hotel theme designed for British players.

## Features

- Responsive design for all devices
- Art Deco / Grand Hotel visual theme
- British-focused content (GBP currency, English language)
- Progressive Web App (PWA) ready
- Modern CSS with custom animations
- Interactive JavaScript features

## Setup

1. Clone the repository
2. Add images to `assets/images/` directory
3. Open `index.html` in a browser

## Notes

- Images need to be added to the `assets/images/` directory
- Replace `{{link}}` placeholders with actual casino links
- Customize colors and styling in `assets/css/style.css` if needed
