# Casoola Casino Landing Page - España

Landing page para Casoola Casino, con tema Art Deco Grand Hotel de los años 30 diseñado para jugadores en España.

## Características

- Diseño responsive para todos los dispositivos
- Tema visual Art Deco / Grand Hotel único
- Contenido completamente en español para jugadores en España
- Moneda: Euro (EUR / €)
- Progressive Web App (PWA) listo
- CSS moderno con animaciones personalizadas
- Funciones JavaScript interactivas

## Configuración Inicial

1. Clonar el repositorio
2. Añadir imágenes al directorio `assets/images/`
3. Abrir `index.html` en un navegador

## Configuración del Repositorio

Este proyecto está conectado al repositorio Casoola_ES:


Para verificar la configuración:

