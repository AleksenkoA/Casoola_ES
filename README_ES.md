# Casoola Casino Landing Page - Versión Española

Landing page para Casoola Casino, con tema Art Deco Grand Hotel diseñado para jugadores españoles.

## Características

- Diseño responsive para todos los dispositivos
- Tema visual Art Deco / Grand Hotel
- Contenido enfocado en España (moneda EUR, idioma español)
- Progressive Web App (PWA) listo
- CSS moderno con animaciones personalizadas
- Funciones JavaScript interactivas

## Configuración

1. Clonar el repositorio
2. Añadir imágenes al directorio `assets/images/`
3. Abrir `index.html` en un navegador

## Conectar al nuevo repositorio

Para conectar este proyecto al nuevo repositorio Casoola_ES, ejecuta:

```bash
git remote set-url origin https://github.com/AleksenkoA/Casoola_ES.git
git remote -v  # Verificar que el cambio se aplicó correctamente
```

Luego puedes hacer commit y push:

```bash
git add .
git commit -m "Versión española del landing page Casoola"
git push -u origin main
```

## Notas

- Las imágenes deben añadirse al directorio `assets/images/`
- Reemplazar los placeholders `{{link}}` con enlaces reales del casino
- Personalizar colores y estilos en `assets/css/style.css` si es necesario
